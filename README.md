# PigLatinApp
Arex Assingment

### Installation API

Install the dependencies and and start the server.

```sh
$ cd piglatinapp.api
$ npm install 
$ mongod
$ npm start
```

### Installation front

Install the dependencies and and start the server.

```sh
$ cd piglatinapp.front
$ npm install 
$ bower install
$ gulp serve
```

### How to use:
  - Create an user
  - Login
  - Start translating words!
